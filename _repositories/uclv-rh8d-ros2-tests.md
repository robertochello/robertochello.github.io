## Overview

This repository collects test examples and validation work for RH8D robotic hand control functionalities using ROS 2.

## Technical Scope

The project is focused on:

- test examples for robotic hand behavior;
- ROS 2 based validation workflows;
- practical experimentation with RH8D control functionalities;
- support material for robotic hand software development.

## Why It Matters

Testing is essential in robotic systems because behavior emerges from the interaction between software, hardware, sensors and the physical environment.

This project supports the validation side of the RH8D robotic hand software ecosystem.
