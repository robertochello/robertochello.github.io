## Overview

This repository hosts my personal technical portfolio.

## Objective

The objective is to maintain a clean and professional web presence for projects, technical notes and professional contact information.

## Technical Scope

The site is built with GitHub Pages and Jekyll. Custom layouts and styles adapt the default theme into a compact engineering portfolio.

## Key Features

- Professional homepage.
- Dedicated About page.
- Project archive based on a custom Jekyll collection.
- Custom repository layout with project metadata.
- CV, GitHub, LinkedIn and email links.

## Status

Active and incrementally maintained.
